# Iris Data Analysis Project

## Overview

This project performs exploratory data analysis (EDA) on the Iris dataset using Python. The objective is to analyze relationships between features using visualization techniques.

## Technologies Used

* Python
* Pandas
* Matplotlib
* Jupyter Notebook

## Dataset Description

The dataset includes the following features:

* Sepal Length
* Sepal Width
* Petal Length
* Petal Width
* Species

## Analysis Performed

* Loaded dataset using Pandas
* Displayed initial records using head()
* Created scatter plot for visualization

## Visualization

A scatter plot is used to study the relationship between sepal length and petal length.

## Key Learning Outcomes

* Understanding of basic data analysis workflow
* Hands-on experience with Pandas and Matplotlib
* Data visualization techniques
